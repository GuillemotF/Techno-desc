# Tech Desc

## Project setup

```
cd back
npm i
```

```
cd front
npm i
```

### Run dev (dev server & serve vue)

```
npm run back
npm run front
```

### Build production

```
npm run build:back
npm run build:front
```

### Run unit tests

```
npm run test:front
```

### Lints and fixes files

```
npm run lint
```
